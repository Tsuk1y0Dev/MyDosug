# Настройка API ключей

## Яндекс.Карты API

Для работы карт необходимо получить API ключ от Яндекс.Карт.

### Шаги настройки:

1. Зарегистрируйтесь на [Яндекс.Разработчик](https://developer.tech.yandex.ru/)
2. Создайте новый проект
3. Получите API ключ для JavaScript API и HTTP Геокодер
4. Создайте файл `.env` в корне проекта:

```env
EXPO_PUBLIC_YANDEX_API_KEY=ваш_ключ_здесь
```

5. Перезапустите приложение

**ВАЖНО**: Никогда не коммитьте файл `.env` в репозиторий! Он уже добавлен в `.gitignore`.

## Бэкенд API

Когда бэкенд будет готов, добавьте URL в `.env`:

```env
EXPO_PUBLIC_API_URL=https://api.mydosug.ru/api
```
